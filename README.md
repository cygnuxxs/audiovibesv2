## ğŸ“¦ Getting Started

### Prerequisites

- Node.js 20.x or higher
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/cygnuxxs/audiovibesv2.git
cd audiovibesv2

# Install dependencies
npm install
# or
yarn install
# or
```
